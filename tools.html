Security Tools

Password Management
Tool Name Source Description
Bitwarden https://bitwarden.com/ This password manager can be used as a cloud based service on multiple devices or hosted locally for more advanced users. It is open source and has passed many audits. It is a great option for a password manager that can be used across devices due to its cloud based nature.
KeePassXC https://keepassxc.org/ This is another open source option; however, it is not a cloud based service. Your password vault will be stored locally so it is important that you make a backup of this. KeePassXC is crossplatform and works with most operating systems both desktop and mobile
A Notebook! N/A Contrary to popular belief, a notebook may just be your best option. So long as you have it with you where you need it and you don't leave it out in the open, it is unlikely that a remote attacker will get access. The threat of someone you know getting access is higher though.
